# Rust Gele!

A hands-on repository for learning Rust step by step.

The examples in this repository follow [The Rust Programming Language](https://doc.rust-lang.org/book/title-page.html). Alongside the exercises from the book, the repository will include small applications that extend those examples and put each concept into practice.

## Goals

- Learn Rust progressively, one concept at a time
- Reinforce each chapter with working examples
- Build practical applications on top of the book's exercises
- Keep the learning journey easy to follow and revisit
